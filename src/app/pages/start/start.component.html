Välkommen!

HB´s Universitet erbjuder 4328 kurser som alla är unika. På hemsidan kan du välja kurser som passar dig och snickra ihop ett eget ramschema, som passar dina önskningar och behov. Behöver du kontakta oss hittar du telefonnumret längst ned på sidan.


Kika runt på hemsidan och känn dig som hemma!

Snabb fakta!

  • 📚 4328 unika kurser och över 60 olika ämnen.
  • 🔍 Filtrera i kurslistan utifrån dina preferenser.
  • ↕️ Sortera utifrån kurskod, kursnamn, poäng eller ämne.
  • 🔗 Länkad kursplan till varje kurs.
  • 🤔 Skapa enkelt ett eget ramschema!
  • 💯 Se totala antal kurser och poäng i ramschemat.
